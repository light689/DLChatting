# DLChatting

### 一种基于 python + ws 的局域网Chat方法

---

### 前言

有的时候要在内网/同机不同用户传递消息，其他平台部署困难，就做了这个。

---

### 使用方法

在服务端输入

```bash
python server.py
```

然后输入

```bash
python client.py
```

即可连接！`注意：如果您打算在客户端连接非localhost的服务器，也就是在不同的电脑连接，需自行在设置进行修改，后期会添加设置进行修改。`
